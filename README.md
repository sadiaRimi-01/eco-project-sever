# EcoTrack Server

This is the backend server for the **EcoTrack** application. It provides APIs for managing eco-challenges, user activities, tips, and events. The server is built with **Node.js**, **Express**, and **MongoDB**, and uses **Firebase Admin** for authentication verification.


## Features
- User registration and management
- Fetch, add, and filter eco-challenges
- Join and track challenges for users
- Fetch tips and upcoming eco-events
- Update user challenge progress
- Secure API access using Firebase authentication
- CORS enabled for frontend communication

## Technologies Used
- **Node.js** – JavaScript runtime
- **Express.js** – Backend framework
- **MongoDB Atlas** – Cloud database
- **Firebase Admin** – Authentication and admin tasks
- **CORS** – Cross-origin requests
- **dotenv** – Environment variable management
